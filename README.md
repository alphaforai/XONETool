# XONETool

📌 项目简介

XONETool 是一款专为 XONE Chain 生态打造的链上工具，致力于为开发者和用户提供 安全、便捷、高效 的区块链工具服务。该项目现已开源，欢迎社区开发者一起共建！

🚀 功能特点

✅ 智能合约管理 - 部署、调用、查询合约

✅ 交易分析 - 解析链上交易数据，提供可视化报告

✅ 区块数据查询 - 轻松获取最新区块信息

✅ 账户管理 - 查询账户余额、交易记录

✅ 开放 API - 提供 RESTful API 供开发者集成
